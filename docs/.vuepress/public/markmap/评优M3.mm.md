# M3 
## （一）体质健康测试成绩（100分）60%
  - 体育教学部提供，大一大二取两学期测试成绩平均分
## （二）日常体育锻炼（20分）
  - 需要提供相关证明，如奖状，有公章的文件，并且截图或者拍照保存备份
  - 1.日常体育锻炼10分，缺勤1次扣1分
  - 2.学校非竞赛类体育活动，报名并完成1分，累计6分
  - 3.数理系非竞赛类体育活动，报名并完成1分，累计4分
## （三）体育赛事分值（20分）

  - 集体项目主力队员加全部分数，非主力队员加分减半
