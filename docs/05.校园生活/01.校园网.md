# 校园网收费

## 北京校区

| 服务对象 | 区域  | 每月计费标准 |
| --- | --- | --- |
| 在校学生（A 类） | 全校  | 0.5 元/天（3 终端） |
|     |     | 0.4 元/G（3 终端） |
|     |     | 10 元 100 小时（1 终端） |
| 驻校企业、商户、   临时人员等（B 类） | 全校  | 60 元/月（2 终端） |

收费依据[规章制度-华电网信处](https://its.ncepu.edu.cn/gzzd/7fe1a3b3ae6c4876b93596ae5e167204.htm)

## 保定校区

同北京校区，公共区域免费（教学楼、食堂），宿舍 0.5 元/天，但是只有一个有线网口。
